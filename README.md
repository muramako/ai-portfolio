# ai-portfolio

AIで生成した、オリジナルキャラクターの画像を紹介するポートフォリオサイト用リポジトリ。GitHub Pagesで公開する。

公開ページ: https://muramako.github.io/ai-portfolio/

## 目的

- 生成済みのオリジナルキャラクター画像を並べたギャラリーサイトを作り、GitHub Pagesで公開する
- 画像の生成自体は行わない。このリポジトリは生成済み画像の公開・展示のみを担当する

## コンテンツポリシー

**オリジナルキャラクターの画像のみ公開する。版権キャラクター（既存作品のファンアート的な生成画像）は公開しない。**

## ディレクトリ構成

- `docs/index.html` — ギャラリーページ（GitHub Pages公開対象）
- `docs/images/` — 公開する画像本体
